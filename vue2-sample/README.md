# vue2サンプルプロジェクト

このプロジェクトはVue2で作成された◯×ゲームのサンプルです。
Vue3が登場する以前のVue2系の標準的なツールセットと書き方で作られています。

## 実行方法

```sh
cd vue2-sample
npm install
npm run serve
```

## 構成

- プロジェクト構築： Vue CLI 4.5.15
- Vue本体バージョン： 2.6.11
- 状態管理: Vuex 3.4.0
- ルーティング: Vue Router 3.2.0
- 言語: TypeScript 4.1.5
- コンポーネントの記法: Options API
